# pasta de imagens

Esta pasta separa a imagem original do resultado gerado pelo projeto.

## images/input

Use esta pasta para colocar a imagem de entrada.

Para executar o script local, salve a imagem com este nome:

```bash
images/input/input.jpg
```

## images/output

O resultado processado sera salvo aqui.

O arquivo gerado pelo script e pelo notebook sera:

```bash
images/output/face_detection_result.jpg
```

Essa imagem deve conter as faces detectadas com bounding boxes e labels simulados.
